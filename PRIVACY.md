# Privacy

Last updated: July 23, 2026

Chaotic Sound Effects does not collect, transmit, sell, or share user data.

The application contains no analytics, advertising, telemetry, accounts, or behavioral tracking.

## Browser extension

The browser extension includes its complete application code, icons, sound index, and audio library. It does not download or execute remote code and does not make runtime requests to external servers.

Browser-managed local storage is used only for the application version and a persistent local copy of the bundled audio pack so the extension can operate offline. This information remains on the user's device and is not transmitted by the extension.

## Progressive web app

The hosted progressive web app is delivered like an ordinary website. The application itself contains no analytics or tracking code and does not set application cookies.

The web host, content-delivery network, and security services may process standard request information—such as an IP address, browser user agent, requested URL, and timestamp—as necessary to deliver and protect the site. Chaotic Sound Effects does not use that operational information to profile or track users.

## External links

The progressive web app and public privacy policy may link to other websites. Visiting an external website is governed by that website's own privacy practices.

## Changes

If this policy changes, the revised policy will be posted at <https://mikeyoung.org/csfx/privacy.html> with an updated date.

## Contact
