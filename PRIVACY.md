# Privacy

Chaotic Sound Effects does not collect, transmit, sell, or share user data.

The browser extension contains its complete application code, icons, sound index, and audio library. It does not use analytics, advertising, telemetry, accounts, cookies, remote code, or runtime network requests. Browser-managed local storage is used only for the application version and a local persistent copy of the static bundled audio pack so the extension can run offline.

The separately hosted progressive web app is served like an ordinary website. Its server may process standard request information needed to deliver the site files, but the application itself contains no analytics or tracking code.
